# 💬 SocialPay

O **SocialPay** é uma ferramenta corporativa que permite enviar e receber pagamentos internos usando `@handles` simples (como redes sociais), preservando a privacidade da organização enquanto mantém um feed auditável de todas as transações.

---

## O Problema

Dentro de organizações, os pagamentos internos e repasses sofrem de:
- **Exposição desnecessária:** Transferências bancárias revelam dados sensíveis (conta, agência, CPF).
- **Falta de rastreabilidade interna:** Não há um feed simples e auditável de "quem pagou quem e por quê".
- **Complexidade:** Sistemas ERP são pesados demais para repasses rápidos entre equipes ou filiais.

## A Solução

O SocialPay cria uma **sub-rede privada** dentro da organização:
1. Cada colaborador ou departamento tem um `@handle` (ex: `@marketing`, `@joao.silva`).
2. Pagamentos são feitos digitando o handle e o valor — sem expor dados bancários.
3. Todas as transações ficam em um **feed auditável** visível apenas para administradores da organização.
4. Os balanços internos são privados (apenas o admin da org vê), enquanto as transações são transparentes dentro do escopo da empresa.

## Casos de Uso

- **Repasse para equipes de campo:** Um gerente envia `50 USDC` para `@equipe-norte` para cobrir despesas operacionais.
- **Bonificações internas:** RH envia recompensas instantâneas via `@handle` sem processos burocráticos.
- **Auditoria simplificada:** O CFO acessa o feed e vê em tempo real todos os movimentos internos.

## Arquitetura

```
Remetente digita @handle + valor
       ↓
  SocialPay resolve handle → Carteira Stellar
       ↓
  Kivo Pay processa transação na Rede Stellar
       ↓
  Feed atualiza via SSE (Server-Sent Events)
       ↓
  Destinatário recebe notificação
```

## Diferenciais

| Característica | SocialPay | Transferência Bancária |
|---|---|---|
| Privacidade | Handle abstrai dados pessoais | Expõe conta, agência, CPF |
| Velocidade | 3-5 segundos | Até 1 dia útil |
| Auditoria | Feed em tempo real | Extrato mensal fragmentado |
| Custo | Fração de centavo | Taxas TED/DOC |
