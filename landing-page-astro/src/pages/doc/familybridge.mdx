# 🏠 FamilyBridge

O **FamilyBridge** é o módulo de remessas internacionais do ecossistema. Ele resolve o problema de milhões de famílias que enviam dinheiro do exterior para o Brasil pagando taxas abusivas.

---

## O Problema

- Taxas de **5–10% por envio** via casas de câmbio tradicionais.
- Câmbio ruim com spreads ocultos embutidos na cotação.
- Demora de **1 a 3 dias úteis** para o dinheiro chegar.
- Experiência péssima para quem não tem conta bancária no exterior.

## A Solução

1. O remetente (EUA, Europa, etc.) informa o valor e o destinatário.
2. O sistema exibe a cotação em tempo real (USDC → BRL).
3. O remetente paga em USDC via carteira Stellar.
4. A rede Stellar processa em **3–5 segundos**.
5. O beneficiário no Brasil recebe via **Pix instantâneo**.

## Integração com o Ecossistema

O FamilyBridge delega toda a ponte de stablecoins e conversão via Pix para o **Kivo Pay**, utilizando:
- **Path Payment Strict Receive** para conversão on-the-fly na rede Stellar.
- **Âncoras SEP-24** parceiras no Brasil para o off-ramp (conversão USDC → BRL → Pix).

## Impacto Social

- Taxas até **90% menores** que remessas tradicionais.
- Dinheiro chega em **segundos**, não dias.
- Comprovante imutável na blockchain para ambas as partes.
