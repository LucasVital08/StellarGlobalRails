# 🤝 Vakinha Global

O **Vakinha Global** é o módulo de financiamento coletivo do ecossistema. Ele permite que organizações e indivíduos criem campanhas de arrecadação onde **cada centavo é rastreável na blockchain**, eliminando a desconfiança que afasta doadores de causas legítimas.

---

## O Problema

- Plataformas tradicionais de vaquinha cobram **taxas de 5-15%** sobre o valor arrecadado.
- Doadores não têm **visibilidade** sobre como o dinheiro é utilizado.
- Campanhas internacionais enfrentam barreiras de câmbio e regulação.

## A Solução

1. A organização cria uma campanha com meta e prazo.
2. Doadores contribuem de qualquer lugar do mundo via USDC.
3. Cada doação é registrada na rede Stellar — **publicamente verificável**.
4. O repasse para o beneficiário é feito via Kivo Pay com rastreio completo.
5. A organização publica relatórios de uso com links para as transações na blockchain.

## Diferenciais

| Vaquinha Tradicional | Vakinha Global |
|---|---|
| Taxa de 5-15% | Taxa mínima (custo de rede Stellar) |
| Sem rastreio de uso | Cada transação é verificável on-chain |
| Apenas moeda local | Doações globais em USDC |
| Relatórios manuais | Dashboard com auditoria em tempo real |

## Impacto Social

A Vakinha Global é especialmente poderosa para:
- **ONGs e projetos sociais** que precisam prestar contas a doadores internacionais.
- **Campanhas emergenciais** (desastres, saúde) onde transparência gera mais doações.
- **Comunidades** que querem financiar projetos locais sem intermediários opacos.
