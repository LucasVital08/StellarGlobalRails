# 🎨 Design System: AI Guidelines

Este documento orienta a IA sobre como manter a consistência visual do ecossistema Stellar Global Rails.

---

## Identidade Visual

### Tipografia (Fontes)
- **Heading:** `Bricolage Grotesque` (Para títulos impactantes).
- **Body:** `Inter` (Para legibilidade).
- **Mono:** `JetBrains Mono` (Para códigos e dados técnicos).

### Cores (Tailwind Palette)
- **Primary:** `Emerald-500` (#10b981).
- **Background:** `Neutral-950` (#0a0a0a).
- **Surface:** `Neutral-900` (#171717).
- **Border:** `White/5` ou `White/10`.

---

## Componentes Padrão (React)

Ao criar novos componentes, siga estes princípios:
1. **Glassmorphism:** Use `backdrop-blur-xl` e fundos com opacidade (ex: `bg-white/5`).
2. **Bordas Sutis:** Bordas devem ser quase invisíveis (`border-white/5`).
3. **Animações:** Use `framer-motion` para transições suaves.
4. **Dark Mode:** O sistema é **Dark-Only** por padrão.

---

## Exemplo de Card Padrão (Tailwind)

```tsx
<div className="p-6 rounded-2xl bg-neutral-900 border border-white/5 hover:border-emerald-500/30 transition-all duration-300 group">
  <div className="w-10 h-10 rounded-lg bg-emerald-500/10 flex items-center justify-center mb-4 group-hover:scale-110 transition-transform">
    {/* Icon here */}
  </div>
  <h3 className="text-xl font-bricolage font-bold text-white mb-2">Título do Card</h3>
  <p className="text-white/60 text-sm leading-relaxed">
    Descrição curta e concisa do benefício ou funcionalidade.
  </p>
</div>
```

## Regras para a IA
1. **Responsividade:** Sempre use classes mobile-first.
2. **Sem Placeholders:** Use ícones da biblioteca `lucide-react` ou `solar` (Iconify).
3. **Acessibilidade:** Garanta contraste suficiente entre o texto e o fundo neutral-950.
