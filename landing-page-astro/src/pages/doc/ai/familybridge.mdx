# 🏠 Specs: FamilyBridge

O FamilyBridge é uma interface especializada sobre o motor do Kivo Pay para remessas EUA/Global → Brasil.

---

## Fluxo de Operação (Core)

1. **Simulação:**
   - Input: `USD`
   - Cálculo: `USDC/BRL` spread + taxa de âncora.
   - API: `GET /payouts/quote` (Kivo API).

2. **Pagamento (On-Ramp):**
   - O usuário envia `USDC` (ou usa SEP-24 para converter Cartão → USDC).

3. **Liquidação (Off-Ramp):**
   - O motor SGR detecta `USDC` na conta de liquidação.
   - Dispara operação via âncora SEP-24 brasileira.
   - Endpoint: `POST /transactions/payout` (API da Âncora).

---

## Componentes Técnicos

### Path Payment (Stellar DEX)
Usado para converter `USDC` para `BRL-token` de forma atômica se houver liquidez direta em ofertas na DEX.

```typescript
const op = StellarSdk.Operation.pathPaymentStrictReceive({
  sendAsset: USDC_ASSET,
  sendMax: maxUsdcAmount,
  destination: receiverAnchorAddress,
  destAsset: BRL_TOKEN_ASSET,
  destAmount: exactBrlAmount,
  path: [] // O SDK encontra o melhor caminho
});
```

## Regras para a IA
1. **Spread:** Sempre arredondar a cotação para baixo em favor do usuário, com margem de segurança de 0.5% para oscilações de liquidez.
2. **Pix:** O `receiver_id` na âncora deve ser a chave Pix do destinatário.
3. **Compliance:** Verificar limites de remessa por CPF antes de autorizar a transação.
