# API Reference

ContractEase has no custom REST backend. All data operations go through the **Supabase JS client** (`@supabase/supabase-js` v2). Privileged operations (Stellar anchoring, OTP delivery) are handled by **Supabase Edge Functions** invoked via `supabase.functions.invoke()`.

---

## Supabase Client Setup

```typescript
// src/lib/supabase.ts
import { createClient } from '@supabase/supabase-js'
import type { Database } from '../types/supabase'

const supabaseUrl = import.meta.env.VITE_SUPABASE_URL
const supabaseAnonKey = import.meta.env.VITE_SUPABASE_ANON_KEY

export const supabase = createClient<Database>(supabaseUrl, supabaseAnonKey, {
  auth: {
    autoRefreshToken: true,
    persistSession: true,
    detectSessionInUrl: true,
  },
})
```

The `Database` generic type is generated by the Supabase CLI:

```bash
npx supabase gen types typescript --project-id fpmhcjkadgebcexzkayz > src/types/supabase.ts
```

## Authentication

### Send Magic Link / OTP

```typescript
// Magic link (for app login)
const { error } = await supabase.auth.signInWithOtp({
  email: 'user@company.com',
  options: {
    emailRedirectTo: `${window.location.origin}/auth/callback`,
    shouldCreateUser: true,
  },
})

// OTP verification (user enters 6-digit code from email)
const { data, error } = await supabase.auth.verifyOtp({
  email: 'user@company.com',
  token: '123456',
  type: 'email',
})
```

### Get Current Session

```typescript
const { data: { session }, error } = await supabase.auth.getSession()
// session.user.id — current user UUID
// session.access_token — JWT for authenticated requests
```

### Listen for Auth Changes

```typescript
const { data: { subscription } } = supabase.auth.onAuthStateChange(
  (event, session) => {
    // event: 'SIGNED_IN' | 'SIGNED_OUT' | 'TOKEN_REFRESHED' | 'USER_UPDATED'
    useAuthStore.getState().setSession(session)
  }
)

// Cleanup
subscription.unsubscribe()
```

### Sign Out

```typescript
await supabase.auth.signOut()
```

## Contracts CRUD

### List Contracts (org-scoped)

```typescript
const { data: contracts, error } = await supabase
  .from('contracts')
  .select(`
    id,
    title,
    status,
    anchor_status,
    stellar_tx_id,
    created_at,
    updated_at,
    created_by,
    contract_parties (
      id, email, full_name, status, role
    )
  `)
  .eq('organization_id', orgId)
  .order('created_at', { ascending: false })
  .limit(50)
```

### Filter Contracts by Status

```typescript
const { data } = await supabase
  .from('contracts')
  .select('*')
  .eq('organization_id', orgId)
  .in('status', ['pending', 'partial'])
  .order('updated_at', { ascending: false })
```

### Get Single Contract with Full Detail

```typescript
const { data: contract, error } = await supabase
  .from('contracts')
  .select(`
    *,
    contract_parties (*),
    contract_clauses (*, order_index),
    contract_logs (action, actor_email, metadata, created_at)
  `)
  .eq('id', contractId)
  .single()
```

### Create Contract

```typescript
const { data, error } = await supabase
  .from('contracts')
  .insert({
    organization_id: orgId,
    title: 'Service Agreement — Acme Corp',
    content: '<p>Contract body...</p>',
    status: 'draft',
    created_by: userId,
    template_id: templateId ?? null,
  })
  .select()
  .single()
```

### Update Contract

```typescript
const { error } = await supabase
  .from('contracts')
  .update({
    title: newTitle,
    content: newContent,
    status: 'pending',
    updated_at: new Date().toISOString(),
  })
  .eq('id', contractId)
  .eq('organization_id', orgId)  // belt-and-suspenders org scoping
```

### Delete Contract

```typescript
const { error } = await supabase
  .from('contracts')
  .delete()
  .eq('id', contractId)
  .eq('organization_id', orgId)
```

## Storage

### Upload Contract PDF

```typescript
const pdfBlob = await generatePDF(contract)  // returns Blob

const filePath = `${orgId}/${contractId}/contract.pdf`

const { data, error } = await supabase.storage
  .from('contract-pdfs')
  .upload(filePath, pdfBlob, {
    contentType: 'application/pdf',
    upsert: true,
  })
```

### Generate Signed Download URL

```typescript
const { data, error } = await supabase.storage
  .from('contract-pdfs')
  .createSignedUrl(`${orgId}/${contractId}/contract.pdf`, 3600) // 1 hour expiry

// data.signedUrl — share with signatories
```

### Upload Signature Image

```typescript
// SignatureCanvas outputs base64; convert to Blob before upload
const blob = await fetch(signatureDataUrl).then(r => r.blob())

  .from('signatures')
```

## Realtime Subscriptions

### Subscribe to Contract Status Changes

```typescript
const channel = supabase
  .channel(`contract-${contractId}`)
  .on(
    'postgres_changes',
    {
      event: 'UPDATE',
      schema: 'public',
      table: 'contracts',
      filter: `id=eq.${contractId}`,
    },
    (payload) => {
      const updated = payload.new as Contract
      setContract(updated)
      if (updated.status === 'completed') {
        toast.success('All parties have signed!')
      }
    }
  )
  .subscribe()

// Cleanup on component unmount
return () => supabase.removeChannel(channel)
```

### Subscribe to Party Signing Events

```typescript
const channel = supabase
  .channel(`parties-${contractId}`)
  .on(
    'postgres_changes',
    {
      event: 'UPDATE',
      schema: 'public',
      table: 'contract_parties',
      filter: `contract_id=eq.${contractId}`,
    },
    (payload) => {
      const party = payload.new as ContractParty
      updatePartyStatus(party.id, party.status)
    }
  )
  .subscribe()
```

## Edge Functions

Edge Functions are deployed to Supabase and called from the client with `supabase.functions.invoke()`. They run in the Deno runtime and have access to the service role key for privileged DB operations.

### `anchor-on-stellar`

Writes the contract SHA-256 hash to a Stellar transaction memo.

```typescript
const { data, error } = await supabase.functions.invoke('anchor-on-stellar', {
  body: {
    contractId: contract.id,
    sha256Hash: contract.sha256_hash,  // 64-char hex string
    network: 'testnet',                // 'testnet' | 'mainnet'
  },
})

// Response:
// { success: true, txId: 'abc123...', explorerUrl: 'https://...' }
```

**Edge Function signature (Deno):**

```typescript
// supabase/functions/anchor-on-stellar/index.ts
Deno.serve(async (req) => {
  const { contractId, sha256Hash, network } = await req.json()

  const server = new StellarSdk.Horizon.Server(
    network === 'mainnet'
      ? 'https://horizon.stellar.org'
      : 'https://horizon-testnet.stellar.org'
  )

  const keypair = StellarSdk.Keypair.fromSecret(Deno.env.get('STELLAR_SECRET_KEY')!)
  const account = await server.loadAccount(keypair.publicKey())

  const hashBytes = Buffer.from(sha256Hash, 'hex')
  const tx = new StellarSdk.TransactionBuilder(account, {
    fee: StellarSdk.BASE_FEE,
    networkPassphrase: network === 'mainnet'
      ? StellarSdk.Networks.PUBLIC
      : StellarSdk.Networks.TESTNET,
  })
    .addOperation(StellarSdk.Operation.payment({
      destination: keypair.publicKey(),
      asset: StellarSdk.Asset.native(),
      amount: '0.0000001',
    }))
    .addMemo(StellarSdk.Memo.hash(hashBytes))
    .setTimeout(30)
    .build()

  tx.sign(keypair)
  const result = await server.submitTransaction(tx)

  // Update contract record with tx ID
  await supabaseAdmin.from('contracts').update({
    stellar_tx_id: result.hash,
    anchor_status: 'anchored',
    anchored_at: new Date().toISOString(),
  }).eq('id', contractId)

  return new Response(JSON.stringify({
    success: true,
    txId: result.hash,
    explorerUrl: `https://stellar.expert/explorer/${network}/tx/${result.hash}`,
  }), { headers: { 'Content-Type': 'application/json' } })
})
```

### `send-signature-email`

Sends a signing invitation email with OTP and magic link.

```typescript
const { data, error } = await supabase.functions.invoke('send-signature-email', {
  body: {
    contractId: contract.id,
    contractTitle: contract.title,
    partyId: party.id,
    partyEmail: party.email,
    partyName: party.full_name,
    signToken: party.sign_token,
    senderName: currentUser.full_name,
  },
})
```

### `validate-otp`

Validates a signing OTP on the server side (prevents client-side bypasses).

```typescript
const { data, error } = await supabase.functions.invoke('validate-otp', {
  body: {
    email: signatoryEmail,
    code: enteredCode,
    contractId: contract.id,
    partyId: party.id,
  },
})

// Response:
// { valid: true } or { valid: false, reason: 'expired' | 'invalid' | 'already_used' }
```

## TypeScript Types

```typescript
// src/types/contract.ts
export type ContractStatus =
  | 'draft'
  | 'pending'
  | 'partial'
  | 'completed'
  | 'cancelled'
  | 'failed'

export type PartyStatus = 'pending' | 'signed' | 'rejected'
export type PartyRole = 'signer' | 'witness' | 'approver'
export type SigningMethod = 'otp' | 'freighter'

export interface Contract {
  id: string
  organization_id: string
  title: string
  content: string | null
  status: ContractStatus
  sha256_hash: string | null
  stellar_tx_id: string | null
  stellar_network: string
  anchor_status: 'pending' | 'anchored' | 'failed'
  anchored_at: string | null
  pdf_url: string | null
  created_by: string
  created_at: string
  updated_at: string
  contract_parties?: ContractParty[]
}

export interface ContractParty {
  id: string
  contract_id: string
  email: string
  full_name: string | null
  role: PartyRole
  status: PartyStatus
  signing_method: SigningMethod | null
  stellar_address: string | null
  signed_at: string | null
  sign_token: string | null
  token_expires_at: string | null
}
```
