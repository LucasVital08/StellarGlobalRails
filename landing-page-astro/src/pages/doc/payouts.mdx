# 💸 Stellar Payouts

O **Stellar Payouts** é o módulo de pagamentos em massa do ecossistema. Ele automatiza o envio de valores para múltiplos destinatários de uma só vez — ideal para folha de pagamento, repasses a fornecedores e distribuição de comissões.

---

## O Problema

- Empresas perdem **horas** fazendo TED/PIX individual para dezenas de fornecedores.
- Pagamentos internacionais em lote são caros e burocráticos via Swift.
- Falta de comprovante unificado: cada pagamento gera um recibo separado.

## A Solução

1. O admin faz upload de uma planilha com destinatários e valores.
2. O sistema agrupa todas as transações em um **Transaction Envelope** (lote Stellar).
3. O Kivo Pay processa o lote inteiro em uma única operação na rede.
4. Cada destinatário recebe o valor e um comprovante individual.

## Casos de Uso

- **Folha de pagamento** para equipes remotas internacionais.
- **Repasse de comissões** para afiliados e parceiros.
- **Distribuição de prêmios** em programas de incentivo.
- **Pagamento de fornecedores** em múltiplos países simultaneamente.

## Vantagens

| Tradicional | Stellar Payouts |
|---|---|
| 1 TED por fornecedor | 1 lote para todos |
| R$10-20 por TED | Fração de centavo por transação |
| Demora variável | Liquidação em segundos |
| Sem rastreio unificado | Dashboard com status de cada pagamento |
