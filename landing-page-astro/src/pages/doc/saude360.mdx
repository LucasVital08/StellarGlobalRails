# 🏥 Saúde 360 Data Wallet

O **Saúde 360** é um protocolo de soberania de dados. Ele permite que pacientes controlem quem acessa seu histórico médico e recebam pagamentos em USDC quando decidem compartilhar seus dados (desidentificados) para pesquisas clínicas ou laboratórios.

---

## O Problema

- **Dados Valiosos, Lucro de Terceiros:** Instituições vendem dados de pacientes para pesquisas sem que o paciente receba nada por isso.
- **Falta de Controle:** O paciente não sabe quem acessou seus exames ou histórico.
- **Silos de Dados:** Seus dados ficam presos em hospitais específicos, dificultando a portabilidade.

## A Solução

1. **Custódia de Dados:** O paciente mantém seus dados cifrados (AES-256) em sua "Data Wallet".
2. **Consentimento On-chain:** As permissões de acesso são registradas como NFTs não-transferíveis ou permissões via Soroban.
3. **Marketplace de Dados:** Pesquisadores solicitam acesso a conjuntos de dados para estudos.
4. **Troca Atômica:** O pagamento em USDC é liberado para o paciente no exato momento em que a chave de decriptografia é fornecida ao pesquisador via Smart Contract.

## Arquitetura Zero-Trust

Os dados médicos **nunca** vão para a blockchain.
- **Off-chain:** Armazenamento cifrado dos documentos médicos.
- **On-chain:** Registro de identidades descentralizadas (DID) e chaves de acesso.
- **Liquidação:** Uso do Kivo Pay para garantir que o paciente receba o valor acordado instantaneamente.

## Benefícios

- **Monetização:** Gere renda passiva com seus dados clínicos históricos.
- **Privacidade Extrema:** Revogação instantânea de acesso — mude de ideia e as chaves são invalidadas.
- **Portabilidade Total:** Leve seu histórico médico completo para qualquer médico em qualquer lugar do mundo.
