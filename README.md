# Stellar Global Rails

Monorepo para produtos e módulos do ecossistema Stellar Global Rails.

## Estrutura

- `apps/`: aplicações de produto (frontend/backend)
- `packages/`: bibliotecas compartilhadas (futuro)
- `docs/`: documentação técnica e de produto (futuro)

## Produtos atuais

- `apps/quilovolt-global-pay`: MVP de pagamentos instantâneos para mobilidade elétrica usando infraestrutura Stellar.
