# Database Schema

ContractEase uses Supabase (PostgreSQL 15) with Row Level Security for multi-tenant data isolation. All tables belong to the `public` schema. The `organization_id` foreign key on every resource table is the foundation of the multi-tenancy model.

---

## Multi-Tenancy Pattern

Every user belongs to one or more organizations via `organization_members`. Every resource (contracts, folders, templates) carries an `organization_id` column. RLS policies use the following helper function to enforce isolation:

```sql
-- Helper: returns the organization IDs the current user belongs to
create or replace function auth.user_organization_ids()
returns uuid[]
language sql stable security definer
as $$
  select array_agg(organization_id)
  from organization_members
  where user_id = auth.uid()
$$;
```

Policies then filter with `organization_id = any(auth.user_organization_ids())`, ensuring that even if the client sends a crafted query, PostgreSQL silently filters to only the caller's organizations.

## Core Tables

### `profiles`

Extends `auth.users` with application-level user data.

```sql
create table profiles (
  id            uuid primary key references auth.users(id) on delete cascade,
  email         text not null,
  full_name     text,
  avatar_url    text,
  phone         text,
  role          text not null default 'user',         -- 'admin' | 'user'
  created_at    timestamptz not null default now(),
  updated_at    timestamptz not null default now()
);

-- Auto-create profile on auth.users insert
create or replace function handle_new_user()
returns trigger language plpgsql security definer as $$
begin
  insert into profiles (id, email, full_name)
  values (new.id, new.email, new.raw_user_meta_data->>'full_name');
  return new;
end;
$$;

create trigger on_auth_user_created
  after insert on auth.users
  for each row execute procedure handle_new_user();
```

### `organizations`

Top-level tenant entity.

```sql
create table organizations (
  id            uuid primary key default gen_random_uuid(),
  name          text not null,
  slug          text unique not null,               -- URL-safe identifier
  cnpj          text,                               -- Brazilian company tax ID
  logo_url      text,
  plan          text not null default 'free',       -- 'free' | 'starter' | 'pro'
  owner_id      uuid not null references profiles(id),
  created_at    timestamptz not null default now(),
  updated_at    timestamptz not null default now()
);

create index idx_organizations_owner on organizations(owner_id);
```

### `organization_members`

Junction table linking users to organizations with a role.

```sql
create table organization_members (
  id              uuid primary key default gen_random_uuid(),
  organization_id uuid not null references organizations(id) on delete cascade,
  user_id         uuid not null references profiles(id) on delete cascade,
  role            text not null default 'member',  -- 'owner' | 'admin' | 'member' | 'viewer'
  invited_by      uuid references profiles(id),
  accepted_at     timestamptz,
  created_at      timestamptz not null default now(),
  unique(organization_id, user_id)
);

create index idx_org_members_user on organization_members(user_id);
create index idx_org_members_org  on organization_members(organization_id);
```

### `contracts`

Central table. Each row is one contract document.

```sql
create table contracts (
  id              uuid primary key default gen_random_uuid(),
  organization_id uuid not null references organizations(id) on delete cascade,
  folder_id       uuid references folders(id) on delete set null,
  template_id     uuid references templates(id) on delete set null,
  title           text not null,
  content         text,                             -- HTML/Markdown body
  status          text not null default 'draft',
    -- 'draft' | 'pending' | 'partial' | 'completed' | 'cancelled' | 'failed'
  sha256_hash     text,                             -- computed after all signatures
  stellar_tx_id   text,                             -- Stellar transaction hash
  stellar_network text default 'testnet',
  anchor_status   text default 'pending',           -- 'pending' | 'anchored' | 'failed'
  anchored_at     timestamptz,
  pdf_url         text,                             -- Supabase Storage path
  created_by      uuid not null references profiles(id),
  created_at      timestamptz not null default now(),
  updated_at      timestamptz not null default now()
);

create index idx_contracts_org        on contracts(organization_id);
create index idx_contracts_status     on contracts(status);
create index idx_contracts_created_at on contracts(created_at desc);
create index idx_contracts_folder     on contracts(folder_id);
```

### `contract_parties`

Every person who must sign a contract.

```sql
create table contract_parties (
  id              uuid primary key default gen_random_uuid(),
  contract_id     uuid not null references contracts(id) on delete cascade,
  organization_id uuid not null references organizations(id) on delete cascade,
  email           text not null,
  full_name       text,
  role            text not null default 'signer',  -- 'signer' | 'witness' | 'approver'
  status          text not null default 'pending', -- 'pending' | 'signed' | 'rejected'
  signing_method  text,                            -- 'otp' | 'freighter'
  stellar_address text,                            -- if freighter signing
  signature_data  text,                            -- base64 image or font name
  signed_at       timestamptz,
  sign_token      text unique,                     -- UUID token in signing link
  token_expires_at timestamptz,
  ip_address      inet,
  user_agent      text,
  created_at      timestamptz not null default now()
);

create index idx_parties_contract on contract_parties(contract_id);
create index idx_parties_email    on contract_parties(email);
create index idx_parties_token    on contract_parties(sign_token);
```

### `contract_clauses`

Optional structured clauses within a contract.

```sql
create table contract_clauses (
  id          uuid primary key default gen_random_uuid(),
  contract_id uuid not null references contracts(id) on delete cascade,
  title       text not null,
  content     text not null,
  order_index int not null default 0,
  clause_type text default 'standard',  -- 'standard' | 'custom' | 'soroban'
  created_at  timestamptz not null default now()
);

create index idx_clauses_contract on contract_clauses(contract_id);
```

### `contract_logs`

Immutable audit trail for every contract event.

```sql
create table contract_logs (
  id          uuid primary key default gen_random_uuid(),
  contract_id uuid not null references contracts(id) on delete cascade,
  actor_id    uuid references profiles(id),
  actor_email text,
  action      text not null,
    -- 'created' | 'sent' | 'viewed' | 'signed' | 'rejected' | 'anchored' | 'cancelled'
  metadata    jsonb,                    -- arbitrary action context
  ip_address  inet,
  created_at  timestamptz not null default now()
);

create index idx_logs_contract    on contract_logs(contract_id);
create index idx_logs_created_at  on contract_logs(created_at desc);
```

### `otps`

Short-lived 6-digit codes for contract signing verification.

```sql
create table otps (
  id          uuid primary key default gen_random_uuid(),
  email       text not null,
  code        text not null,                -- 6-digit string (hashed in production)
  contract_id uuid references contracts(id) on delete cascade,
  party_id    uuid references contract_parties(id) on delete cascade,
  used        boolean not null default false,
  expires_at  timestamptz not null default (now() + interval '15 minutes'),
  created_at  timestamptz not null default now()
);

create index idx_otps_email      on otps(email);
create index idx_otps_expires_at on otps(expires_at);
```

### `folders`

Organizational hierarchy for contract filing.

```sql
create table folders (
  id              uuid primary key default gen_random_uuid(),
  organization_id uuid not null references organizations(id) on delete cascade,
  parent_id       uuid references folders(id) on delete cascade,
  name            text not null,
  created_by      uuid references profiles(id),
  created_at      timestamptz not null default now()
);

create index idx_folders_org    on folders(organization_id);
create index idx_folders_parent on folders(parent_id);
```

### `templates`

Pre-built contract templates (52 shipped with seed data).

```sql
create table templates (
  id              uuid primary key default gen_random_uuid(),
  organization_id uuid references organizations(id) on delete cascade,
    -- null = global template (visible to all orgs)
  title           text not null,
  description     text,
  content         text not null,
  category        text,                   -- 'nda' | 'service' | 'employment' | 'real-estate' | ...
  language        text default 'pt-BR',
  is_public       boolean default false,
  usage_count     int default 0,
  created_by      uuid references profiles(id),
  created_at      timestamptz not null default now(),
  updated_at      timestamptz not null default now()
);

create index idx_templates_org      on templates(organization_id);
create index idx_templates_category on templates(category);
create index idx_templates_public   on templates(is_public) where is_public = true;
```

### `notifications`

In-app notification inbox.

```sql
create table notifications (
  id          uuid primary key default gen_random_uuid(),
  user_id     uuid not null references profiles(id) on delete cascade,
  type        text not null,             -- 'contract_signed' | 'contract_anchored' | 'invite' | ...
  title       text not null,
  body        text,
  link        text,
  read        boolean not null default false,
  metadata    jsonb,
  created_at  timestamptz not null default now()
);

create index idx_notifications_user on notifications(user_id);
create index idx_notifications_read on notifications(user_id, read) where read = false;
```

## Row Level Security Policies

### `profiles` RLS

```sql
alter table profiles enable row level security;

-- Users can read their own profile
create policy "profiles_select_own"
  on profiles for select
  using (id = auth.uid());

-- Users can update their own profile
create policy "profiles_update_own"
  on profiles for update
  using (id = auth.uid());

-- Org admins can read profiles of their org members
create policy "profiles_select_org_members"
  on profiles for select
  using (
    id in (
      select user_id from organization_members
      where organization_id = any(auth.user_organization_ids())
    )
  );
```

### `organizations` RLS

```sql
alter table organizations enable row level security;

create policy "orgs_select_members"
  on organizations for select
  using (id = any(auth.user_organization_ids()));

create policy "orgs_insert_authenticated"
  on organizations for insert
  with check (auth.uid() is not null);

create policy "orgs_update_owner_admin"
  on organizations for update
  using (
    id in (
      select organization_id from organization_members
      where user_id = auth.uid()
        and role in ('owner', 'admin')
    )
  );
```

### `contracts` RLS

```sql
alter table contracts enable row level security;

create policy "contracts_select_org"
  on contracts for select
  using (organization_id = any(auth.user_organization_ids()));

create policy "contracts_insert_org"
  on contracts for insert
  with check (
    organization_id = any(auth.user_organization_ids())
    and created_by = auth.uid()
  );

create policy "contracts_update_org"
  on contracts for update
  using (organization_id = any(auth.user_organization_ids()));

create policy "contracts_delete_owner"
  on contracts for delete
  using (
    organization_id in (
      select organization_id from organization_members
      where user_id = auth.uid() and role in ('owner', 'admin')
    )
  );
```

### `contract_parties` RLS

```sql
alter table contract_parties enable row level security;

-- Org members can see all parties on their contracts
create policy "parties_select_org"
  on contract_parties for select
  using (organization_id = any(auth.user_organization_ids()));

-- Public signing page access: party can see their own row via sign_token
-- (handled via anon key + service role in Edge Function)

create policy "parties_insert_org"
  on contract_parties for insert
  with check (organization_id = any(auth.user_organization_ids()));

create policy "parties_update_org"
  on contract_parties for update
  using (organization_id = any(auth.user_organization_ids()));
```

## Storage Buckets

```sql
-- Contract PDFs (private, org-scoped)
insert into storage.buckets (id, name, public) values ('contract-pdfs', 'contract-pdfs', false);

-- Signature images (private)
insert into storage.buckets (id, name, public) values ('signatures', 'signatures', false);

-- Storage RLS: users can only access files in their org's folder
create policy "contract_pdfs_org_access"
  on storage.objects for select
  using (
    bucket_id = 'contract-pdfs'
    and (storage.foldername(name))[1] = any(
      select id::text from organizations
      where id = any(auth.user_organization_ids())
    )
  );
```
