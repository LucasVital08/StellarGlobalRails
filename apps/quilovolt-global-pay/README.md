# QuiloVolt Global Pay

MVP demonstrável de pagamentos instantâneos para mobilidade elétrica dentro do ecossistema Stellar Global Rails.

## Escopo do MVP

- Landing institucional do produto
- Demo interativa do fluxo de recarga
- Simulação de pagamento via Stellar (USDC/BRZ)
- Liberação simulada de sessão de recarga
- Comprovante digital da operação
- Dashboard do operador com métricas e sessões

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Lucide React
- Recharts

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.
